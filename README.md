# Лабораторная работа по прикладному программированию

Цель: отработать работу с системой контроля версий Git через командную строку.
Репозиторий демонстрирует ветвление, Git Flow, rebase, cherry-pick, откаты, хуки и теги.
## Структура репозитория

- main — стабильная ветка с релизами.
- develop — основная ветка разработки.
- feature/* — временные ветки для отдельных задач лабораторной работы.
